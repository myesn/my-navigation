# 简介

几年前写的一个简洁的静态导航单页面，支持一级导航和二级导航。

有两个版本，分别是：
- [my-navigation.7z](my-navigation.7z): 不要求输入密码
- [my-navigation-with-authentication.7z](my-navigation-with-authentication.7z): 要求输入密码
